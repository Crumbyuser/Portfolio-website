Personal Budget Tracker

Technologies: Python, Tkinter
Duration: 2 months
Role: Full Stack Developer
View on GitHub

A desktop application built with Python and Tkinter that helps users track their personal finances. The application features a user-friendly interface for managing expenses and income, with the following key features:

  • Expense tracking and categorization
  • Income management
  • Budget planning and monitoring
  • Simple and intuitive GUI using Tkinter
  • Data persistence for long-term tracking

The project demonstrates my ability to create practical desktop applications using Python and implement user-friendly interfaces with Tkinter. The application helps users maintain better control over their finances through an easy-to-use interface.
